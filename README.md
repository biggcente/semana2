# Página de Agentes de Valorant

Este proyecto es una página web simple hecha en HTML que muestra información sobre algunos agentes del juego Valorant.

## Contenido
- Estructura con HTML5 (header, nav, article, aside)
- Menú de navegación con enlaces internos
- Información de agentes:
- Imágenes de cada personaje

## Estilos
Se utiliza un archivo externo estilo.css para el diseño de la página.

## Funcionalidad
- Navegación entre secciones
- Visualización de agentes y sus descripciones
